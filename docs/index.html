What it does

Three arts, one extension

📋

Clipboard History

Every text you copy, kept. Search, pin, tag, edit, and export items. Survives screen lock, Shell restarts, and logouts.

🪄

Spells

Select a clipboard item, cast a spell, get the transformed result as a new entry. The original is never touched.

🧙

Aldric

A persistent floating chat. Streaming replies, multiple conversations saved to disk. Runs entirely on your machine.

Getting started

Install in three steps

i. fetch
$ npm install
ii. build
$ npm run install-extension
iii. enable
$ gnome-extensions enable grimoire@iammanu.dev

After enabling, reload GNOME Shell — Xorg: Alt+F2 → r; Wayland: log out and back in.
Open Settings from the popup to configure your Ollama host and pick a model.
